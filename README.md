# Invoice Management System - Skill Exam

This is a Spring Boot REST API for managing Invoices.

## Endpoints
- **POST**: `http://localhost:8080/api/invoices/add` - Add a new invoice.
- **GET**: `http://localhost:8080/api/invoices/viewall` - View all saved invoices.
